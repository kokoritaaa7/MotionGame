Folder Arrange

Unzip the archive and find out the folder "buyer-file". There are all the template files in this folder. You can open the following files for viewing in browser.

  1. for all resources
    • css — folder with css files.
    • fonts — folder with icon fonts.
    • images — folder with image files.
    • js — folder with Javascript files.
  2. index.html is for Home Page
  3. about-us.html About page
  4. cart.html Cart Page
  5. check-out.html Check Out Page
  6. features.html Features Page
  7. login.html Login Page
  8. profile.html Profile Page
  9. registration.html Registration Page
  10. shop.html Shop Page
  11. shop-details.html Shop Details Page
  12. tournaments.html Tournaments Page
  13. tournaments-single.html Tournaments Single Page
  14. error.html Error Page
  15. contact.html Contact Page

Logo Type

To change logotype in the navigation panel, find image images/logo.png and replace it with your logo.

Note: Keep your logo inside the images/logo.png folder. The logo size should be minimum 163x53 pixel.
